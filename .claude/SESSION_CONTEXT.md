## Critical conventions & gotchas

1. **Read `node_modules/next/dist/docs/` before writing framework code.** Next.js 16 has breaking changes. AGENTS.md is the canonical reminder.
2. **`searchParams` is async** in Next.js 16 — type as `Promise<...>` and `await sp` (see `src/app/inquire/page.tsx` for the pattern).
3. **Iframe `sandbox` is forbidden on /book** — Square detects sandboxed contexts and silently refuses to navigate. Same caveat applies if you ever wrap the Pic-Time iframe.
4. **Content is a single source of truth.** Add a new service: (1) `ServiceSlug` union in `src/lib/types.ts`, (2) `services` array in `src/lib/content.ts`, (3) `portfolios` array, (4) `KNOWN_SLUGS` in `scripts/import-photos.mjs`, (5) optional `makesOffer` entry in `src/app/layout.tsx`. Sitemap, JSON-LD, and `generateStaticParams` derive automatically.
5. **Add a new questionnaire:** define a `Questionnaire` constant in `src/lib/questionnaires.ts` (most start with `yourDetailsSection` and `bookingStatusSection(slug)` as sections 1–2), register it in the `QUESTIONNAIRES` map. Service page CTA, sitemap, and index page pick it up automatically.
6. **Visibility lives in one place.** `evaluateShowIf` and `visibleSectionsFor` (`src/lib/questionnaires.ts:~1947`) are used by both `<QuestionnaireForm>` and `/api/questionnaire/route.ts`. Don't duplicate the logic — call the helper.
7. **`showIf` semantics:**
   - `equals` / `equalsAny` → hidden until parent is answered.
   - `notEquals` / `notEqualsAny` → visible when parent is unset (so scope-narrowing rules don't blank the form before the user picks a package).
8. **The `package` field type** is auto-resolved at render-time from `services[slug].packages` via `resolvePackageOptions(slug)`. Never hard-code package option lists.
9. **Cross-page link copy** uses `[label](href)` markdown inside `comboNote` / `pricingNote` / intro strings. The `renderInline()` helper in `src/lib/inline.tsx` handles it.
10. **Google Reviews degrade gracefully.** If `GOOGLE_PLACES_API_KEY` or `GOOGLE_PLACE_ID` are unset, `<GoogleReviews>` returns `null`. Don't add error states for this.
11. **Date awareness.** Today is **2026-04-10** as of the last session. Convert relative dates to absolute when writing memory.
